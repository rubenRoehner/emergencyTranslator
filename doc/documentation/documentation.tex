\begin{algorithm}
\caption{Bubble sort algorithm.}
\begin{algorithmic}

\STATE $S$ is an array of integer
\FOR {$i$ in $1:(length(S)-1)$}
    \FOR {$j$ in $(i+1):length(S)$ }
        \IF {$S[i]> S[j]$}
            \STATE swap $S[i]$ and $S[j]$
        \ENDIF
    \ENDFOR
\ENDFOR

\end{algorithmic}
\label{bubble_sort}
\end{algorithm}


The Bubblesort algorithm is shown in Algorithm \ref{bubble_sort}.
